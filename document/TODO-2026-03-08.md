
`terraform.tfvars`에서 `api_server_authorized_ips`를 현재 IP로 갱신:
```hcl
api_server_authorized_ips = ["<현재 공인 IP>/32"]
```

## Phase 2: Addon 설치

> **설치 순서 고정 (의존성 기반)**
> install.sh가 자동 실행하지만, 수동 실행 시 아래 순서 준수.

### 00. Priority Classes (전제 조건)
```bash
bash addons/scripts/00-priority-classes.sh
```

### 00b. Gateway API CRD
```bash
bash addons/scripts/00b-gateway-api.sh
```
- v1.5.0 설치 (주석의 v1.3.0과 다름 — 코드 기준이 맞음)

### 01. cert-manager
```bash
bash addons/scripts/01-cert-manager.sh
```
- ClusterIssuer (staging + production) 자동 생성 포함
- `LETSENCRYPT_EMAIL` 설정 시 HTTP-01 fallback 자동 적용

### 02. External Secrets Operator
```bash
bash addons/scripts/02-external-secrets.sh
```

### 04. Istio (설치)
```bash
bash addons/scripts/04-istio.sh
```

### 04b. Istio mTLS STRICT 설정 (핵심)
```bash
bash addons/scripts/04b-istio-mtls.sh
```
- 네임스페이스 레이블(`istio.io/rev=asm-1-28`) 자동 적용 포함
- PeerAuthentication STRICT 자동 적용 포함
- DestinationRule ISTIO_MUTUAL 자동 적용 포함

### 05. Kyverno
```bash
bash addons/scripts/05-kyverno.sh
```

### 06. Flux GitOps
```bash
bash addons/scripts/06-flux.sh
```
- Extension 설치 + FluxConfig 생성 자동 포함
- `GITOPS_REPO_URL` 설정 시 SSH 키 기반 GitOps sync 자동 시작

### 07. Kiali
```bash
bash addons/scripts/07-kiali.sh
```
- Operator 설치 + Kiali CR 생성 자동 포함
- Prometheus 연동: `PROMETHEUS_URL` 환경변수로 자동 구성

### 08. Karpenter NodePool
```bash
bash addons/scripts/08-karpenter-nodepool.sh mgmt
bash addons/scripts/08-karpenter-nodepool.sh app1
bash addons/scripts/08-karpenter-nodepool.sh app2
```
- capacity-type: `spot` 전용 (on-demand 포함 시 ADR-007 위반)
- NodePool 적용 후 워크로드 배포 시 Spot 노드 자동 프로비저닝 확인

## Phase 3: 최종 점검

### 3-1. 전체 검증 스크립트 재실행

### 3-2. mTLS STRICT 적용 확인
```bash
# 각 클러스터에서 실행
kubectl exec -n <namespace> <pod> -- curl -v http://<service>:80
# TLS handshake 로그가 보이면 mTLS 동작 중
```

### 3-3. 클러스터 간 통신 테스트
```bash
# mgmt → app1, mgmt → app2 통신 확인
# post-apply-check.sh Section 11에서 자동 검증됨
```

### 3-4. Backup 상태 확인
```bash
# ProtectionError → ProtectionConfigured로 변경 확인
az dataprotection backup-instance list \
  --resource-group rg-k8s-common \
  --vault-name bvault-k8s \
  --query "[].{name:name, status:properties.currentProtectionState}" \
  -o table
```

## 알려진 이슈 및 주의사항

| 항목 | 내용 | 조치 |
|------|------|------|
| AKS Free Tier | Control Plane SLA 없음 | 서비스 트래픽 전 Standard 전환 |
| ACR Basic | 이미지 공개 인터넷 경유 | 운영 시 Standard + PE 전환 |
| Authorized IP | 공인 IP 변경 시 kubectl 차단 | 매일 IP 확인 후 tfvars 갱신 |
| Grafana 비활성화 | 대시보드 없음 | 필요 시 enable_grafana = true |
| Flux FluxConfig | 06-flux.sh에서 자동 생성 | GITOPS_REPO_URL 필수 설정 |
| Kiali CR | 07-kiali.sh에서 자동 생성 | PROMETHEUS_URL 미설정 시 메트릭 비활성 |
| mTLS PeerAuthentication | 04b에서 자동 적용 | 기존 파드는 rollout restart 필요 |
